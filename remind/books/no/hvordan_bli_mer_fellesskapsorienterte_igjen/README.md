# Hvordan blir vi mer fellesskapsorienterte igjen?

Ny lokal ReMind-bokserie i markdown-format.

Serien er laget på bakgrunn av spørsmålet:

"Hvordan kan vi gjøre samfunn, foreldre og oss alle mer fellesskapsorienterte igjen? Det individfokuserte samfunnet er vel ikke så bra for oss?"

## Bøker

1. `bok_1_bordet_som_ble_storre_barn.md`
   - Barnebok med tre historier.
   - Tema: deling, lek, ansvar, å se den som faller utenfor, og at fellesskap ikke betyr at alle må være like.

2. `bok_2_ikke_bare_min_profil_ungdom.md`
   - Ungdomsbok.
   - Tema: prestasjon, sosiale medier, gruppepress, ensomhet, vennskap, lokal handling og digitalt ansvar.

3. `bok_3_fra_jeg_til_vi_voksne.md`
   - Voksenbok/lærebok.
   - Tema: individ, fellesskap, foreldre, institusjoner, marked, omsorg, grenser, demokrati og felles praksis.

## Kjerne

Serien kritiserer ikke individet som sådan.

Mennesker trenger frihet, grenser, egen stemme og personlig ansvar.

Men et samfunn som gjør mennesket til et selvprosjekt uten sterk nok tilhørighet, omsorg og felles ansvar, gjør både barn og voksne mer utrygge, mer ensomme og lettere å styre gjennom skam, prestasjon og forbruk.

Seriens kjerne er:

- fellesskap uten selvutslettelse
- frihet med ansvar
- foreldre som ikke skal bære alt alene
- barn som ikke skal være små prosjekter
- nabolag, skole og institusjoner som øver på felles ansvar
- grenser som beskytter fellesskapet
- rettferdighet som mer enn snillhet
- digital kultur som ikke gjør mennesket til merkevare
- å bygge små konkrete praksiser, ikke bare store idealer
- å spørre: Hva trenger vi nå, og hvem bærer kostnaden hvis ingen svarer?

Dette er ikke publisert materiale, bare lokale `.md`-utkast.
