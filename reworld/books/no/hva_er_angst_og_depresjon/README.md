## Viktig

Dette er psykoedukasjon og refleksjon, ikke diagnose eller behandling.

Ved akutt fare, selvmordstanker, selvskading eller alvorlig funksjonsfall må man kontakte kvalifisert hjelp, legevakt eller nødtjeneste.

## Kjerne

- Angst forstås som alarm som kan bli for sterk, for ofte eller for styrende.
- Depresjon forstås som mørke, tyngde, tap av glede og håp som kan bli alvorlig.
- Barn og ungdom skal ikke være behandlere for hverandre.
- Venner kan være lykter, men voksne og helsepersonell må inn når det trengs.
