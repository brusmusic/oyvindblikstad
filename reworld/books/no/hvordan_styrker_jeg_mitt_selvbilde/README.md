# Hvordan styrker jeg mitt selvbilde?

Lokal ReMind-bokserie i markdown-format.

## Bøker

1. `bok_1_speilet_som_lurte_barn.md`
2. `bok_2_filteret_ungdom.md`
3. `bok_3_et_sannere_bilde_voksne.md`

## Kjerne

Selvbildet styrkes ikke ved å gjøre bildet perfekt, men ved å gjøre det sannere, mildere og mer forankret i verdighet.

Serien undersøker:

- speil og sammenligning
- indre språk
- kroppens påvirkning
- gamle historier
- skam og ansvar
- relasjoner som korrigerer eller forvrenger
- handling i tråd med verdier
